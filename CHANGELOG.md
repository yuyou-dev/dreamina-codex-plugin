# Changelog

## 0.1.1

- 同步 2026-06-22 版 `dreamina` CLI 帮助。
- 新增图片模型 `4.7` 支持：`text2image`、`image2image`。
- 新增视频模型 `seedance2.0mini` 支持：`text2video`、`image2video`、`frames2video`、`multimodal2video`。

## 0.1.0

- 初始开源版本。
- 支持 Dreamina / 即梦本地 CLI 的图片生成、视频生成、任务查询、积分查询和登录会话管理封装。
- 默认中文服务。
- 使用 MIT 协议发布。
